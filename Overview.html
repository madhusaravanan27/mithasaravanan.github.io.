Program Overview

Program Learning Outcomes

The master’s in Applied Data Science program is interdisciplinary, offering students a comprehensive education in data science fundamentals and advanced applications. The program aims to develop students' skills in:

  • Data Collection, Storage, and Access: Collecting, storing, and accessing data by identifying and leveraging applicable technologies.
  • Creating Actionable Insights: Creating actionable insights across a range of contexts (societal, business, political) using the full data science lifecycle.
  • Visualization and Predictive Modeling: Applying visualization and predictive models to generate actionable insights.
  • Programming Proficiency: Utilizing programming languages like R and Python to support data analysis.
  • Communication of Insights: Communicating insights via visualization and analytics to a broad audience, including both technical and non-technical stakeholders.
  • Ethics in Data Science: Applying ethical principles in data science, such as fairness, bias reduction, transparency, and privacy.

High-Level Project Descriptions

Throughout the program, I completed a range of projects that solidified my understanding of data science and gave me hands-on experience with industry-standard tools:

  • VBAY Database Data Warehouse Project: Developed a data warehouse to centralize and optimize VBAY's data processes across business areas, including sales and bidding. This project leveraged tools like DBT and Snowflake to create dimension and fact tables, supporting efficient data analysis.
  • Crime Classification Based on Geographical Area: Analyzed crime patterns in Los Angeles using machine learning and data visualization to uncover insights into crime hotspots, demographics, and temporal patterns. Models such as RandomForest and XGBoost were used to predict crime occurrences, helping stakeholders make informed decisions.
  • Visual Exploration of Natural Disasters and Economic Impact (2000-2020): Explored the frequency, severity, and economic impact of natural disasters across the United States through interactive dashboards. Using tools like Tableau, this project provided stakeholders with actionable insights into disaster preparedness and recovery efforts.

Achievement of Program Learning Outcomes

Each project allowed me to achieve specific program learning outcomes:

  • Data Collection, Storage, and Access: Mastered through the VBAY Data Warehouse Project, where I learned efficient data storage and retrieval techniques using Snowflake and DBT.
  • Creating Actionable Insight: Realized in the Crime Classification project, which translated raw crime data into insights relevant to various stakeholders, from law enforcement to residents.
  • Visualization and Predictive Modeling: Demonstrated in the Natural Disasters project through advanced visualization and predictive analysis to make insights accessible and actionable.
  • Programming Proficiency: Strengthened through projects using Python and R, applying data transformation, analysis, and visualization techniques.
  • Communicating Insights: Practiced in each project, especially through interactive dashboards and visual reports that convey complex insights to non-expert audiences.
  • Ethical Data Use: Upheld across all projects, ensuring unbiased and respectful data handling, particularly in sensitive data contexts like crime analysis.
